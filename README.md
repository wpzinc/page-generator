# Page Generator WordPress Plugin

Welcome to the Page Generator WordPress Plugin repository on GitHub. Here you can view the plugin source code, review open issues, and see releases of the plugin.

**If you're not a developer, please download the plugin from [WordPress.org](https://wordpress.org/plugins/page-generator/). Downloading and installing directly from GitHub will not work.**

## Support

If you are having issues setting up this plugin on your WordPress site, or have issues with Page Generator, please do the following:

* Check the [Documentation](https://www.wpzinc.com/documentation/page-generator-pro)
* Review WordPress.org's [Support Forum](https://wordpress.org/support/plugin/page-generator/)

Paid license holders may request [email support](https://www.wpzinc.com/support)
